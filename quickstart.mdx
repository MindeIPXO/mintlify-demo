# Quickstart

## Install Mintlify CLI

```bash
npm i -g mintlify
```

## Run locally

```bash
mintlify dev

# browse at locally at http://localhost:3000/
```

## Validate OpenAPI specs

```bash
mint openapi-check openapi/billing.openapi.yaml
mint openapi-check openapi/inventory.openapi.yaml
```
